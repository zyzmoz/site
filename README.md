# My Portfolio

React App Created to Serve my Portifolio!

## Tools used
- React
- Redux
- Parcel Bundler

## Commands

Install all dependencies:
```
  yarn install
```

Run development server (port 8000):
```
  yarn start
```

Run build (default folder 'dist'):
```
  yarn build
```
